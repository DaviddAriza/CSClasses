README:
Open the “CS324E Final Project.pde” file from the folder.
Hover over ‘Sketch’ at the top of the IDE, hover over ‘Import Library’,  then click on ‘Add Library’.
Search for ‘Minim’
Ensure that this library is installed by either seeing a green checkmark to the left of the library name, or by pressing install at the bottom of the window once clicking on this library
Click run to load the game. The main menu will be displayed.
Click the tutorial button to access the tutorial page and read the instructions of the game.
Select a game mode and customize the fall rate and number of asteroids using the sliders.
Click “Play” to start the game.
Click “Pause” or hit the SPACE key to pause during gameplay.
